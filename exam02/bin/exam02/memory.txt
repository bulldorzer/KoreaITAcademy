
# 메모리 공간 

1) 코드 영역 - 프로그램 코드
2) 데이터 영역 
- 프로그램 시작부터 종료까지 계속 유지되는 변수
ex) static int num = 10; // 데이터 영역에 위치

3) 스택 영역 
-  선언했을때 생성, {}중괄호 끝나면 소멸
- 일반적인 변수, 지역변수, 매개변수

ex) int num = 10; // 스택영역

4) 힙 영역

- new 클래스명(); // 힙영역
- new 키워드를 통하여 생성된 메모리
  ㄴ 메모리의 주소값을 menu 변수에 넣어준다.
  ㄴ menu 변수에는 힙 영역에 할당된 주소값이 들어있다.
  
클래스명 참조변수명 = new 클래스명();
       -> 주소값

Menu menu = new Menu();

Menu menu;  ------------> 스택
menu = new Menu(); -----> 힙


일반자료형 - int, double, char, boolean
			ㄴ 값을 직접 가지고 있음
			
참조자료형- 주소값을 가지고 있는 변수
			클래스명 변수명; 
			인터페이스 변수명;
			
		String 변수 = "문자열"; // 리터럴 상수의 주소값 
				ㄴ 해당 문자열의 주소값을 가지고 있음.

		// 다른 공간에 "문자열"이라는 글자가 새롭게 할당된다.
		String 변수 = new String("문자열");
